# CRM Frontend

React + Vite frontend for Huisartsen CRM.

## Development Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

The dev server runs on http://localhost:3000 and proxies API calls to the backend at http://localhost:5000.

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **React Router** - Client-side routing
- **Axios** - HTTP client
- **Lucide React** - Icons
